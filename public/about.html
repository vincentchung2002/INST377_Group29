About CityBike Finder

Our Mission

CityBike Finder is a comprehensive web application designed to revolutionize urban mobility by providing real-time information about bike-sharing networks across different cities.

Key Features

  • Comprehensive Bike Network Search
  • Real-Time Station Information
  • Detailed Bike Availability Tracking
  • User-Friendly Interface

Data Sources

Our application aggregates data from:

  • CityBikes API
  • Supabase Database

Our Team

Developed by INST377 Group 29, a team of passionate students dedicated to creating innovative urban mobility solutions.
