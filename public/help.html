Search Networks

Use the search bars at the top of the page to find bike networks:

  • Network Name Search: Type the name of a bike network
  • Location Search: Enter a city or country name
  • You can use both search bars together to refine your results

Viewing Station Details

üö≤

Network Stations

To view stations for a specific network:

  • Click on a network card in the main list
  • Station details will appear below the network list
  • Each station card shows:
    • Station Name
    • Number of Free Bikes
    • Number of Empty Slots
    • Last Updated Timestamp
